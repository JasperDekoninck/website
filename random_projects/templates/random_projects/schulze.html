Schulze

This program helps you calculate the result of a vote. The method used to do this, is called Schulze. More information about this method can be found on this Wikipedia page.

Add options

Invalid input. See explanation below on how to add valid votes.

Result based on current votes

A vote can be entered as follows: a '-', '_' or '' can be used to differentiate options (indicated by their letter) (eg A-B). To point out that there is no preference between two options, just write them one after the other (eg AB). Options that appear last don't need to be entered. The program adds these options automatically (AC -> AC-BD). You can press enter to add a new vote.
